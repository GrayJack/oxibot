# OxiBot

A bot in Rust

## Features
 - Add caller to roles
 - Remove caller to roles
 - Get latency of shard
 - Get running system uname and uptime
 - Fortune messages
 - Shrug, tableflip and unflip lennys

## Minimal Viable Rust Version
Rust 1.37

## Requirements
 - uname
 - uptime
 - fortune-mod
